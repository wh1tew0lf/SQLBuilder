# SQLBuilder
Classes for generate SQL queries (select/insert/update/delete) on different platforms (MySQL, MS SQL)

### For run tests:
`./vendor/bin/phpunit`

### For create documentation:
`./vendor/bin/phpdoc -d ./src -t ./docs/`
